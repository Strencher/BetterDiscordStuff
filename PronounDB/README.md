# PronounDB
Shows you the pronoun of a user right next to their name. Pronouns by https://pronoundb.org

### Features
- Pronouns from [PronounDB](https://pronoundb.org)
- Option to prefer Discord Pronouns over PronounDB
- Custom Pronouns
